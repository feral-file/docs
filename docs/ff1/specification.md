# FF1 Specification

| Item | Spec |
|------|------|
| **Dimensions** | Approx. 128 × 128 × 42 mm vertical enclosure |
| **CPU** | AMD Ryzen 7 7735U (8 cores, 16 threads, up to 4.75 GHz) |
| **GPU** | Integrated AMD Radeon 680M graphics (hardware-accelerated 4K decode) |
| **RAM** | 8 GB DDR5 4800 MHz (upgradable via two SO-DIMM slots, up to 64 GB) |
| **Storage** | 128 GB NVMe SSD (M.2 2280, PCIe 4.0; user-replaceable and expandable) |
| **Display** | HDMI 2.0 (4K @ 60 Hz included); additional displays via DP 1.4 and USB-C display output on the internal M700 |
| **Connectivity** | Wi-Fi 5 (802.11ac, 2.4/5 GHz), Bluetooth 5.0, Gigabit Ethernet, 2 × USB 3.2, 2 × USB 2.0 |
| **Modularity** | Sealed for simplicity (HDMI + power exposed by default); removable mesh side panels unlock full I/O (USB, Ethernet, DP/USB-C, audio) and future expansions |
| **Power** | 19 V DC adapter (included), 65 W; designed for 24/7 operation within normal indoor temperature ranges |
| **Enclosure** | Black-anodized aluminum finned lid and bottom (extruded 6061 top + CNC 6061 base) with 3D-printed ABS mesh side panels and rubber feet; 1 mm thermal pad couples to the metal chassis to the lid for ~15–28 W sustained loads |
| **OS** | FF OS (Linux-based, deterministic WebGL/WebAssembly rendering) |
| **Onboarding** | QR-code pairing via mobile app; typical setup in under five minutes |
| **Updates** | Automatic over-the-air (OTA) system and player updates |
| **Blockchain Support** | Ethereum (ERC-721, ERC-1155) and Tezos (FA2) collections, with on-chain ownership used for playback rights |
| **Media Formats** | Images (JPG, PNG, SVG, GIF); Video (MP4 H.264/H.265, WebM VP9, MOV); Audio (AAC, MP3, OGG) |
| **Code Execution** | HTML/CSS/JS sandboxed in Chromium, with WebGL/WebGL2, WebAssembly, and OffscreenCanvas for real-time and generative works |
| **Warranty** | 1-year global hardware warranty; user-serviceable internals (RAM/SSD access) |
